Building the AI Operating Layer

This was a big week, maybe the biggest one yet. I went from having a dashboard to having an actual system that runs the companies, with a team of AI “employees” doing the repetitive work in the background. I'm genuinely proud of what I shipped.

Where I hit walls, and what I changed

Being honest about the hard parts — that's where I learned the most.

›
The scheduling almost bit me. I first had Pete run on the built-in scheduler, and I caught that it runs late and skips. The lesson was clear: never trust that as the real trigger. I moved everything to a reliable external timer — the same way the other bots run — and made the report idempotent so it can never send twice.
›
Cal wasn't catching Mike's requests. Mike asked Leah to order something in the group and it never became a task. Two problems: the bot only reacted to keyword phrases, and privacy settings meant it couldn't even see normal messages. So I rebuilt it to read the room, with an AI gate that decides whether a message is actually a task before filing it, and I turned the privacy setting off so it can see the conversation.
›
Revoking a token quietly broke a bot. When I rotated a token, it wiped the bot's connection without warning. I learned I have to re-register that connection every time — and I did.
›
The demo asked for a password on mobile when it shouldn't have. It worked on desktop but not on my phone. The cause was that mobile browsers behave differently, so a couple of files were getting blocked. I fixed the specific files and hardened the whole gate so only real page visits ever ask for a login, never background assets.
›
My calendar isn't connected yet. It needs an admin setting turned on at the account level. Rather than wait, I built a working mockup of my week so the page still feels complete — and it'll swap to live the moment that setting is enabled.
›
I pushed the design harder when the first versions felt flat. I wasn't happy with a couple of passes and reworked them until they actually looked like something I'm proud to show.
Where we're at

The foundation is real and running. The bots handle the daily and weekly load, the reporting reaches Mike on a predictable rhythm, and everything is documented so it can scale to the next company without starting over. This week turned a nice-looking dashboard into an operating system.
